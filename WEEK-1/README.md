##  Programs Included

| File Name | Description |
|------------|-------------|
| array_search_function.js | Searches an element in an array and returns its index |
| array_sum_function.js | Finds the sum of array elements using a function |
| arrsum.js | Calculates the total sum of array elements |
| biggest_of_three_numbers.js | Finds the biggest among three numbers using ternary operator |
| biggest_of_two_numbers.js | Finds the biggest among two numbers using if-else |
| employee_array_operations.js | Performs insert, delete, and update operations on employee array objects |
| largest_of_three_fun.js | Finds the largest among three numbers using function |
| small_element_in_arr.js | Finds the smallest element in an array |

---

##  How to Run the Programs

1. Install Node.js
2. Open terminal in the project folder
3. Run the program using:

```bash
node filename.js
```

### Example

```bash
node array_search_function.js
```
