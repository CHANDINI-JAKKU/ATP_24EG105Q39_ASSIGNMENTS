# Week 4 HTML Assignments

## Description
This repository contains Week 4 HTML assignments that demonstrate the basics of web page creation using HTML elements such as headings, paragraphs, lists, links, images, and nested lists.

---

## Objectives
- Learn the structure of HTML documents
- Understand headings, paragraphs, and lists
- Practice creating links and inserting images
- Build simple webpages using HTML

---

##  Technologies Used
- HTML5
- VS Code
- Web Browser
