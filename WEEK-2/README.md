#  JavaScript Mini Projects & Programs

##  Description
This repository contains multiple JavaScript mini projects and practice programs developed to understand core JavaScript concepts such as arrays, objects, functions, validation, cart systems, task management, and data manipulation.

---

##  Objectives
- Practice JavaScript fundamentals
- Understand array and object operations
- Build mini projects using JavaScript
- Improve problem-solving and logical thinking
- Learn modular programming concepts

##  Technologies Used

- Node.js
- VS Code

##  How to Run the Programs

1. Install Node.js
2. Open terminal in the project folder
3. Run any JavaScript file using:

```bash
node filename.js
```

### Example

```bash
node otp.js
```
