# Week 5 Projects

## Overview
This folder contains Week 5 web projects focused on layout design, React practice apps, tables, and Tailwind CSS.

## Projects Included

### 1. Flexbox Layout
- `flex.html` and `flex.css`
- Demonstrates layout design using CSS Flexbox
- Includes responsive alignment, spacing, and content distribution examples

### 2. Products React App
- `products_react_app/`
- A React-based project displaying a product catalog using mock data.
- **Key Features**: Clean component structure (`ProductList` -> `Product`) and separate data files for modularity.

### 3. Users React App
- `users_react_app/`
- A React-based project displaying a list of user profiles.
- **Key Features**: Modular component architecture (`UsersList` -> `User`) with separated mock data.

### 4. Tables
- `Tables/table1.html`, `Tables/table2.html`, `Tables/table3.html`
- `Tables/style1.css`, `Tables/style2.css`, `Tables/style3.css`
- Table examples demonstrating:
  - header and body styling
  - alternate row striping and hover states
  - responsive table layout and border design
  - custom color themes and spacing for improved readability

### 5. Tailwind Project
- `Tailwind-Project/package.json`
- `Tailwind-Project/src/`
- A Tailwind CSS-based project showcasing utility-first styling and responsive design
- Includes examples of:
  - reusable Tailwind classes for spacing, colors, and typography
  - responsive layout utilities for mobile and desktop views
  - component-style structure using Tailwind utilities in HTML/JSX
  - configuration-ready setup for Tailwind in a modern frontend workflow

## How to Use

### Static HTML/CSS Projects
Open the `.html` files directly in a browser to view the layouts and table examples.

### React Projects
If the React apps are configured with a build tool, open the project folders and run the appropriate commands, for example:

```bash
cd WEEK-5/products_react_app
npm install
npm run dev
```

```bash
cd WEEK-5/users_react_app
npm install
npm run dev
```

### Tailwind Project
Install dependencies and run the development server from the Tailwind project folder:

```bash
cd WEEK-5/Tailwind-Project
npm install
npm run dev
```

## Notes
- The `Tables` folder contains multiple styled table examples.
- The React app folders include `src/` directories for application source code.
- Use browser dev tools to inspect layout behavior and responsive styling.
